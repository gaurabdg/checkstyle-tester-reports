19   * This package provides models that simulate certain physical phenomena
20   * of Earth and the near-Earth environment.
21   * <p>
22   * Currently the following models are included:
23   *
24   * <ul>
25   *   <li>Tropospheric Delay</li>
26   *   <li>Geomagnetic Field</li>
27   *   <li>Geoid</li>
28   * </ul>
29   *
30   * <h2>Geoid</h2>
31   *
32   * <p>A Geoid is an equipotential surface of Earth's gravity field. This package
33   * provides the means to compute a Geoid from the gravity field harmonic
34   * coefficients, as in the following example. See the comment for {@link
35   * org.orekit.models.earth.Geoid} for some important caveats.
36   *
37   * <pre><code class="brush: java">
38   * ReferenceEllipsoid ellipsoid
39   *         = new ReferenceEllilpsoid(a, f, bodyFrame, GM, spin);
40   * Geoid geoid = new Geoid(
41   *         GravityFieldFactory.getNormalizedProvider(degree, order),
42   *         ellipsoid);
43   * double undulation = geoid.getUndulation(lat, lon, date);
44   * </code></pre>
45   *
46   * @author T. Neidhart
47   * @author E. Ward
48   *
49   */
